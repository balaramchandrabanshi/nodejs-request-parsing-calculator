# Node.js Request Parsing Calculator

## About
A beginner Node.js project to practice handling HTTP requests. It adds two numbers while focusing on request parsing and data handling.

## Concepts
- Module import/export
- Reading request data in chunks
- Buffering and parsing data
- Sending HTML response

## Tech
- Node.js
- JavaScript (built-in http module)

## Learnings
- Requests arrive in chunks, need buffering
- Extracting and processing user input
- Understanding backend flow without a framework

## Run
1. Clone the repo
2. Open project folder
3. Run: `node index.js`
4. Open browser at `http://localhost:3000`
